# 🌐 Personal Portfolio Website

A modern and responsive personal portfolio website built using **HTML5**, **CSS3**, **Bootstrap 5**, and **JavaScript**. This portfolio showcases my skills, projects, education, and contact information in a clean and professional design.

## 🚀 Features

- 📱 Fully Responsive Design
- 🎨 Modern UI with Bootstrap 5
- 📌 Sticky Navigation Bar
- 👤 Hero Section with Introduction
- 🙋 About Me Section
- 💻 Skills Section
- 📂 Projects Showcase
- 🎓 Education Timeline
- 📞 Contact Form & Social Links
- ⬆️ Scroll-to-Top Button
- ✨ Smooth Scrolling & Hover Effects

## 🛠️ Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- Bootstrap Icons
